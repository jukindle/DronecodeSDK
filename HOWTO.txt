Install dependencies

1. Update stuff
	sudo apt-get update -y
2. Install stuff
	sudo apt-get install cmake build-essential colordiff astyle git libcurl4-openssl-dev libtinyxml2-dev doxygen -y


Setting up DronecodeSDK

1. Clone the repository
	git clone https://github.com/jukindle/DronecodeSDK.git
2. Change directory
	cd DronecodeSDK
3. Clone all subrepositories
	git submodule update --init --recursive
4. Pull all submodules
	git submodule update --recursive --remote

Download precompiled libraries using Docker

1. Install docker
	sudo apt install dockerio
2. Download stuff
	sudo docker run --rm -it -v $HOME/DronecodeSDK:/root/DronecodeSDK:rw dronecode/dronecode-sdk-ubuntu-16.04 bash
	
Compile the SDK system wide

1. Clean everything
	make clean
2. Compile default stuff
	make default
3. Apply system wide
	sudo make default install
